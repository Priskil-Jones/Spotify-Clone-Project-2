Spotify Clone
A front-end Spotify Web Player clone built using HTML and CSS. This project recreates the look and feel of Spotify's user interface, including the sidebar, playlists, music cards, and player section.

Features:
- Spotify-inspired user interface
- Responsive layout
- Sidebar navigation menu
- Playlist and album cards
- Music player design section
- Modern dark theme
- Hover effects and smooth styling

Technologies Used:
- HTML
- CSS
- Font Awesome Icons

Note:
This project was created for HTML and CSS practice purposes only. It is a static front-end clone and does not include JavaScript or backend functionality. Music playback, search, navigation buttons, and other interactive features are not functional.
